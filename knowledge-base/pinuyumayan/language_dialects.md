# 卑南族語言與方言知識庫 (Language & Dialects)

## 1. 語言概述

卑南語（Puyuma language）屬於臺灣南島語系，是卑南族的母語。由於部落間的地理區隔與歷史發展，卑南語內部存在顯著的方言差異。

## 2. 四大方言群

根據原住民族語言研究發展基金會（ILRDF）與 FormosanBank 的語料庫分類，卑南語主要分為四個方言群，這在語音合成（TTS）與語言教學上必須特別區分：

1. **南王卑南語（Nanwang Puyuma）**：主要通行於南王部落，屬於竹生系統。其語音保留了較多古音，是目前研究最透徹的方言之一。
2. **知本卑南語（Zhiben Puyuma）**：通行於知本部落，屬於石生系統。
3. **建和卑南語（Jianhe Puyuma）**：通行於建和部落，與知本語較為接近。
4. **西群卑南語（Xiqun Puyuma）**：包含初鹿、利嘉等部落的方言。

## 4. 語音與 TTS 處理規範

在開發與卑南語相關的語音合成（TTS）系統時，必須遵循以下規範：

1. **方言支援**：必須支援南王、知本、建和、西群四種方言的區分與切換。
2. **發音規則**：羅馬拼音的發音規則更新時，必須使用英文近似音，禁止使用中文注音描述，以免 AI 誤判。
3. **禁用詞彙**：禁用的詞彙清單中，絕對不能包含簡體中文字。
